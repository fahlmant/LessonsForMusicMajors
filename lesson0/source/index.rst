Downloading Python 3
--------------------

We are using Python 3, and a tool called IDLE.


Follow this `link
<https://www.python.org/ftp/python/3.5.0/python-3.5.0-webinstall.exe>`_
for Windows 7/8/10


Follow this `link
<https://www.python.org/ftp/python/3.5.0/python-3.5.0-macosx10.6.pkg>`_ or Mac OSX

For ubuntu or other linux, we will use the command line

.. code-block:: bash

    sudo apt-get install python3
